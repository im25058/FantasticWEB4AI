Project Name: Fantastic WEB4 AI

Description:
Fantastic WEB4 AI is a multi-agent research system that generates verifiable reports using LangGraph.

Features:
- Planner, Research and Validator agents
- Transparent audit logs
- Cryptographic report hashing
- Streamlit interface

Live Demo:
https://fantasticweb4ai.streamlit.app

Instructions to run:

pip install -r requirements.txt
streamlit run frontend/app.py
